# Library Management System

It is REST API for a Library Management System Project using Spring Boot, Spring Data JPA (Hibernate), and MySQL database.

## Functionalities

* Create a new User
* Fetch all the Users
* Add new book
* Fetch all the books
* Fetch specific book
* Delete a book 
* Borrow a book 
* Return a book

## Technologies Used

* Spring Web, 
* Spring Data JPA, 
* MySQL Driver, 
* and Lombok
